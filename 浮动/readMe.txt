网页会根据块元素或者内联元素的特性，按照从左至右，从上到下的方式自然排序！
        这种元素自然排列的方式，我们称之为====》标准文档流！

浮动 float
  左浮动
  右浮动
浮动之后 会脱离 标准文档流

溢出 overflow
 overflow:
             01.visible:默认显示
             02.scroll：以滚动条的形式显示溢出部分
             03.hidden：溢出部分隐藏
             04.auto：自动跳转
